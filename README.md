Repo with solution for [Backend Position Technical Task](https://gist.github.com/kami4ka/40ad1f66be45bb37cabdb94f9923d721)

## Install
```
npm install
```

## Run
```
npm run dev
```


## Env vars
- PORT - port for server to listen on
- DB - DB name
- TABLE - DB table name
- TTL - time to live for cached values in seconds
